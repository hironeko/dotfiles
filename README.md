# Mydotfiles + prezto : Set Up Guide

- 前提と対象
  - macOSであること
  - zshが存在していること

- 下記コマンドで`Homebrew` + `prezto` の設定が完了する
```shell
curl -sL https://raw.githubusercontent.com/hironeko/dotfiles/master/install/install.sh | sh
```
